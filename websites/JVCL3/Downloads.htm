Downloads

 

The JVCL can be downloaded in several ways depending on whether you want to be on the "bleeding edge" or want to use a more stable version.

 

Official Releases

The official releases of JVCL can be downloaded from the sourceforge project page.

 

Daily zips

Every night a couple of zip files are created automatically containing a snapshot of the current CVS repository. Since these zip files are generated automatically, they might contain errors and the zip file might even be corrupted. To access the daily zips, go to http://jvcl.sf.net/daily to see a listing of available downloads.

 

CVS

To always have access to the most recent changes in JVCL, you should install a CVS client (we recommend TortoiseCVS and WinCVS) and download the CVS repository files to your computer. With the CVS client, you can update your local repository at any time. For more instructions on how to set up CVS and use it with JVCL, see the CVS instruction page. You can also access the CVS repository via the web. Click this link to go to the CVS root, click this link to go to the root folder of current development.

 

Daily tarballs

A daily tarball of the entire CVS repository is available for backup or mirroring purposes at http://cvs.sourceforge.net/cvstarballs/jvcl-cvsroot.tar.gz.
