Roadmap


Below is outlined the roadmap for the next few months. This is a proposition and may need to be adjusted to take into account unforeseen elements.
 

30th August 2004 - 12th September 2004:

  • Mantis Bug fixes. As many bugs as possible must be reviewed, assessed and fixed where applicable. Some of these bugs have been in there for too long. If a bug reporter does not provide feedback within two weeks of answering him, the issue is considered closed.


1st September 2004:

  • Launch of Jedi Help, the online version of the Jedi help, with the JVCL as its first project. Users are invited to contribute.


13th - 14th September 2004:

  • Creation of JVCL 3 Beta 2 and Creation of the associated help file from Jedi Help.


15th September 2004:

  • JVCL 3 Beta 2 is distributed, users can start using it and provide feedback through Mantis. The associated help file is distributed as well, users contribute on Jedi Help.


15th - 30th September 2004:

  • User feedback on JVCL 3 Beta 2 is coming in, bugs are fixed as they are indicated. Mantis is the preferred way of getting notice.


1st - 3rd October 2004:

  • Preparation of JVCL 3 Release Candidate 1, with integrated help file.

4th - 10th October 2004:

  • Feeback from users about RC1. Expected number of bugs is low. If any, they are fixed as they appear, through Mantis.


11th October 2004:

  • Preparation of JVCL 3 Stable.


12th October 2004:

  • Release of JVCL 3 Stable.
