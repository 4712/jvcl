Overview

GLOBUS VCL Extensions Library (GVCL) for Delphi & C++ Builder is a library of components and functions for Borland Delphi 3 - 7 and Borland C++ Builder 5 with source texts of all units. The given library is FreeWare product. You are free to distribute the library without change of source texts of units and contents of install archive. Correctness of operation of components is not guaranteed. The responsibility for possible(probable) damage as a result of usage of the given program code completely is shifted on you. However the author accepts any remarks on incorrect operation of components with the purpose of the prompt elimination of the errors detected by you.

The Brief description

Visual components:

TglBevel

расширение TBevel extension such possibilities, as gradient filling by 5 methods, choice of the displayed sides of a rectangle, mapping of a grid, additional bold drawing style.
пример

TglCheckBox

Allows to set glyphs for states of the switch and the background image. It is sensitive to the mouse: allows to set style (5 variants) and color (the text and a background) for active (when the mouse pointer is above a component) and passive states. It can be displayed as underlined in active state.
пример

TglDigits

Digital board
пример

TglShadow

Wrapper for any window components. Can create effect of a shadow and bevels around of given component, change of style of a border at obtaining / loss of focal point and so forth.
пример

TglFlyingText

Creation of effect of the flying volumetric text wifth gradient filling.
пример

TglGroupBox

inheritance of TCustomGroupBox. Allows to display a border of header and client area (BevelInner, BevelOwter, BevelBold) and gradient filling. Alignment of header, and diffirent text styles (5 variants plus style fgoDelineatedText in Options). At the option fgoCanCollapse components can turn collapse on click on its header. Also can be displayed glyph in the header. Allow to grouping in scroll areas ala 3DS Max.
пример

TglBitmapImage

With help TglBitmapImage the image can be multiplied, proportionally to scale, create the Disabled-image.
пример

TglLabel

5 drawing styles plus mapping of an outline (fgoDelineatedText in св-ве Options). 4 directions of output of the text. Change of style and color at moving above a component of a mouse pointer. Allow draw textured text. It gives the extended possibilities for binding focused (FocusControl) handles. Limitation: output only one-line inscriptions.
пример

TglStaticText
TLabel with possibility of text justify on width plus some more possibilities.

TglListBox

Wordwrap items; customization of style of a items bevels; Allow set Glyphs; Allow fill client area with background image. Allow change color of the font, hottrack option etc.
пример

TglAskListBox

The list to which each recording there corresponds a number of buttons. Pressing of the button derivates event OnButtonClick. The state of buttons: GetPushedButtonInLine (Index:word):integer; - returns number of the selected button in string(line) Index; CountPushedButtonsInColumn (Column: integer):integer; - returns index of selected buttons in column; IsFilled:boolean; - returns true if from all recordings of the button are selected; Possibilities of draw customization are close to TglListBox;
пример

TglTabControl

TglPageControl & TglTabControl. Allows to output bookmarks from any side with independent customization of a direction of the font. For a selected / non-withdrawn bookmark fonts, frameworks (Inner, Outer), a background are customized. Filling by a gradient which it is possible to set for each bookmark individually. Also it is possible to specify in execution time color of the font and a background for each bookmark.
пример

TglHoleShape

Transparent holes in the form with customized 3D boundary.
пример

TglCheckTreeView

TglTreeView & TglCheckTreeView. Allow to set glyphs for checked and unchecked state etc.
пример

TglProgress

Progressbar with gradient filling, customized style and the format mask of an label. Only horizontal variant. COOL:-)
пример

TglRuler
Vertical or horizontal ruler.

glSplit
Cool splitter

TglHelpPanel

The panel placed from below of the form and containing are given the test of hint RTF. It is opened on click. (ala hints in Allaire HomeSite)
пример

TglWizardHeader

Header for creation of wizards. For using with TPageControl ect.
пример

Data aware components:

TglDBGrid
DBGrid with multilower case headers of fields, property Glyphs for headers. Autoalignment of headers on width.

Non-visual components:

TglExceptionHandler
Components of processing and log of exceptions. Saving of the protocol in the file or sending on MailSlot. sources...
Exception Viewer Expert

TglMailSlotServer & TglMailSlotClient
The client and the server for Windows Mailslots - exchange of the text data on the network.
Read article on this subject here (on russian): http://cpr.biblio-globus.ru/mailslots.htm

glSmallFontsDefence
The component protect the application from the distortion at start in the mode of large fonts. It is possible to remove protection with grids to not deprive with users of all advantages of large fonts.
Read article on this subject here (on russian): http://cpr.biblio-globus.ru/small_font_defence.htm

glXMLSerializer
Serialization objects in XML and its loading. Very useful, has integrated parser.
Read article on this subject here (on russian): http://www.delphikingdom.com/mastering/xml_ser.htm

glSingleInstance
Allow to start only one copy of the application

Tgl3DColors

TglCaption

TglHint

TglProcess
The elementary wrapper for start of processes, watchings of their operation and completion.

TglStringContainer
Just TStringList container :)

TglSysInfo

glSysRequirements
Checks at start of the application the version of operating system, presence of memory, screen resolution. Outputs warning and - or unloads the application
